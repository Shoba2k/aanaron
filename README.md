## Quick Start

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser to see the pamphlet.

### Build for Production

```bash
npm run build
```

The static files will be generated in the `dist` folder, ready for deployment.

### Preview Production Build

```bash
npm run preview
```
